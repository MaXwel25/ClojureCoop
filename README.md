# Mire Arena

Мультиплеерная арена, вдохновленная [Mire](https://github.com/technomancy/mire), с использованием Clojure и (пока подумаю:)).

## Описание

Проект представляет собой мультиплеерную игру с графическим интерфейсом, которая представляет собой забавны йчат для общения! Отдельно работает как серверная, так и клиентская часть!
