# Mire Arena

Мультиплеерная арена, вдохновленная [Mire](https://github.com/technomancy/mire), с использованием Clojure 1.11.1 и quil 2.8.0 для графики.

## Описание

Проект представляет собой мультиплеерную игру с графическим интерфейсом, которая представляет собой забавный чат для общения! Отдельно работает как серверная, так и клиентская часть!
